Brainrot Meter

Nothing here leaves your machine. There is no account, no server, and no network code in this extension — the data lives in your browser's local storage and dies with it.

Your goal

The thing the scrolling is taking from. Named, this shows up on the pause screen — and it does far more work than any number ever will.

The pause

Seconds of enforced wait before a feed opens. This is the part that actually changes behaviour. 0 turns it off. Once you continue, that site gets a 15-minute grace period — no nagging mid-scroll.

Check-in

Show a small reminder every N minutes of feed time in a day. 0 turns it off.

Never count these

One fragment per line. Any URL containing one is ignored completely — not tracked, never interrupted.

What gets counted

Only the infinite surfaces. YouTube Shorts counts; a YouTube lecture does not. Reddit's front page, r/popular and r/all count; a subreddit you chose to open does not. If the tool ever calls your real work brainrot, it has failed and you should tell it to stop.
